Изображение #2: Гроссбух мамы
(файл: images/slide-02-grossbuch.png)

Гроссбух мамы (02.03.1976):

  • 1 хлеб
  • 3 л молока
  • 0.5 кг сахара
  • ИТОГО: 1 руб. 42 коп.

Подбор цен:

  • Попытка 1: Всё по 20 коп. → Ошибка!
  • Попытка 2: Молоко 24 коп. → Ошибка!
  • ✓ Найдено!
    Хлеб: 16 коп. | Молоко: 28 коп. | Сахар: 84 коп.

Итоговый расчет:

Новая покупка:
1 хлеб + 0.5 л молока + мороженое

Расчет: 1×16 + 0.5×28 + 1×15 = 45 коп.

✓ Сдачи с полтинника на мороженое хватит!

Как превратить слово в число?

Игра «ДаНетка»:

Слово = «пуговица»

  • Живое? → 0
  • Сделано человеком? → 1
  • Металлическое? → 0.3
  • Для одежды? → 1
  • Для застегивания? → 1

Эмбеддинг пуговицы:

[0, 1, 0.3, 1, 1, 0, ...]

← 2000 чисел в моделях!

Три этапа обучения

1️⃣ Предсказание

Обучение без учителя

«Париж» → «Париж является» →
«Париж является столицей» →
«Париж является столицей Франции» ✓

Все тексты из интернета

2️⃣ Вопрос-ответ

Обучение с учителем

Вопрос: Назови столицу Франции.
Пример: Швеция: Стокгольм

Ответ: Франция: Париж ✓

3️⃣ Оценка ⚠️

Обучение с подкреплением

  • Париж → 3/10
  • Столицей является Париж → 8/10
  • Развернутый ответ → 10/10
  • Грубый ответ → 1/10
❗ Третий этап — самый ответственный! От него зависит полезность и безопасность ИИ.
