## Всё про ИИ для всех

Это лекция для пионеров и пенсионеров, для гуманитариев и технарей, для всех, кто хочет понять, как устроены современные нейросети — без формул, но с уважением к сути.

Здесь нет кода. Нет математики за пределами школьной арифметики. 
Есть волшебная палочка Цыбенко, волшебные очки внимания, игра данетка и Дед Мороз.

А еще здесь есть мой "Мнифест начинающего ИИ-прсветителя", объясняющий, зачем я подготовил эту лекцию.
