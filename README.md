Всё про ИИ для всех

О чём это
Это лекция для пионеров и пенсионеров, для гуманитариев и технарей, для всех, кто хочет понять, как устроены современные нейросети — без формул, но с уважением к сути.

Здесь нет кода. Нет математики за пределами школьной арифметики. Есть волшебные палочки, взмахи и очки внимания.

И есть манифест, объясняющий, зачем я это написал.

Для кого это
Для преподавателей — можно брать целиком или фрагментами

Для любопытствующих — чтобы понять, что происходит у них в телефоне

Для встревоженных — чтобы не впадать в крайности

Как использовать
Читать — просто для себя

Рассказывать — своим студентам, коллегам, соседям

Улучшать — делайте форки, дополняйте, переписывайте

Критиковать — открывайте issues, указывайте на ошибки

Никаких лицензионных ограничений, кроме просьбы: если делаете лучше — делитесь.

Почему это здесь
Я не ищу единомышленников. Я даже предпочёл бы критиков. Я ни к чему не призываю и ничего не пропагандирую.

Я просто сделал то, что мог, в силу своей ответственности и компетенции.

Если вы можете сделать лучше — сделайте.
